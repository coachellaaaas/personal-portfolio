## How it's built

The site is a single `index.html` file — no React, no build step, no dependencies. Just HTML, CSS variables, and a small amount of vanilla JS for the animated SNN diagram. It's deployed on GitHub Pages straight from the `main` branch.

The design uses:
- `Space Grotesk` for body text
- `JetBrains Mono` for labels, tags, and code-style elements
- `Fraunces` for project headings
- A dark theme built with `oklch()` colour space for perceptually consistent colours
